# Project Name

A Node.js/Express application generated with [ServerGen](https://github.com/theinfosecguy/ServerGen).

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm

### Installation

```bash
npm install
```

### Running the Application

```bash
npm start
```

The server will start on http://localhost:3000
